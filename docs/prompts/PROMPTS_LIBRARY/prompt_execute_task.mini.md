# חובה  קובץ זה הוא חלק משיטת Docs as System mini  
# חובה  אין לערוך קובץ זה בתוך פרויקט פעיל
# חובה  שינוי בקובץ זה יתבצע רק בעדכון רשמי של השיטה

# ביצוע משימה  

מטרה  
לבצע את המשימה הנוכחית בצורה מדויקת  
תואמת למסמכי המערכת  
ועומדת בכללי הפיתוח של השיטה

הוראות  
קרא את המשימה מתוך תוכנית המימוש  
בדוק את הקלט הנדרש למשימה  
בדוק אילו קבצים מותרים לעריכה  
בדוק אילו קבצים אסורים לעריכה  
קרא את ההגדרות הרלוונטיות מהמסמכים  
בצע את הפעולה הקטנה ביותר הדרושה כדי להשלים את המשימה  
שמור על עקביות עם כל המסמכים  
הימנע משינויים שאינם חלק מהמשימה  
בצע refactor רק אם מוגדר במפורש  
הכנס בדיקות יחידה חדשות אם המשימה דורשת זאת  
עדכן בדיקות קיימות רק אם נדרש

תנאי המשך  
המשימה בוצעה במלואה  
אין חריגות מהמסמכים  
כל הבדיקות עברו  
אין שינויים מיותרים בקבצים שאינם חלק מהמשימה

פלט מצופה  
שינוי מדויק בקוד בהתאם למשימה  
בדיקות תקינות  
קבצים מעודכנים בתוך תחום המשימה בלבד
